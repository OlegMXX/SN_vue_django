# How to run locally

## To run frontend (inside frontend directory):
### Project setup
```
npm install
```
### Compiles and hot-reloads for development
```
npm run serve
```

## To run backend (inside backend directory):
### Project setup
```
pip install -r requirements.txt
```
### Run development server
```
python manage.py runserver
```
